DitherForge - Getting Started

Because DitherForge is not signed with an Apple Developer certificate, macOS will block it the first time you try to open it. You only need to do this once.

Step 1. Drag ditherforge.app to your Applications folder (or wherever you like).
Step 2. Open Terminal (search for it in Spotlight) and paste this command:

xattr -cr /Applications/ditherforge.app

If you put the app somewhere else, adjust the path accordingly.
Step 3. Double-click ditherforge.app to launch it. It will open normally from now on.
